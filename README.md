# Heart Disease Prediction App

This is a Streamlit-based machine learning web application that predicts the risk of heart disease based on patient health information.

## Features
- User-friendly interface
- Machine learning prediction
- Probability score display
- Real-time prediction

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Joblib
